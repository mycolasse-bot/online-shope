Aurelia Mode - Website Toko Online Fashion Wanita

Cara hosting:
1. Upload semua isi folder fashion-wanita-store ke hosting.
2. Pastikan file index.html berada di root/public_html hosting.
3. Buka domain website Anda.

Catatan:
- Website ini berbasis HTML, CSS, dan JavaScript, jadi bisa dipakai di hosting statis.
- Produk, cart, admin login, dan pesanan demo tersimpan di browser pembeli/admin memakai localStorage.
- Desain dibuat lebih compact dengan gaya marketplace fashion: header putih/abu, search, banner, kategori, flash sale, dan grid produk.
- Tersedia pilihan warna tampilan: Rose, Lavender, Kuning Shopee, Orange, Biru, Sage, Hijau, Mocha, Abu, dan Putih. Pilihan warna juga mengubah header dan tersimpan otomatis di browser.
- Checkout di halaman home sudah dihapus. Kartu produk membuka halaman product.html dengan 6 foto produk nyata, deskripsi, jumlah, total produk otomatis sebelum ongkir, cart, checkout, ekspedisi, dan metode pembayaran.
- Alamat pengiriman di halaman produk dipisah menjadi Nama Jalan, Kelurahan, Kabupaten, dan Provinsi.
- Metode pembayaran demo: transfer bank, QRIS, ShopeePay, GoPay, OVO, DANA, dan COD.
- Untuk toko produksi dengan pembayaran otomatis dan notifikasi real-time WhatsApp/email, hubungkan ke backend atau platform pembayaran.
